# Next.js WhatsApp AI Demo (Arkiton)

Quick demo project containing a single Next.js page with a WhatsApp integration simulation.

## How to run locally

1. Install dependencies:
   npm install

2. Run dev server:
   npm run dev

3. Open http://localhost:3000

## Notes
- The demo uses Tailwind via CDN for quick preview. For production, install Tailwind properly.
- Replace pages/index.js whatsappNumber variable with your real number (country code + number), e.g. '919876543210'.
- Deploy to Vercel by pushing to GitHub and importing the repo.
